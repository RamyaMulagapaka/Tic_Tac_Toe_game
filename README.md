# Tic Tac Toe Game in Java

A simple console-based Tic Tac Toe game developed using Java.

## Features

- Two-player game
- Player name input
- Displays game board after every move
- Detects winner
- Detects draw
- Prevents invalid moves
- User-friendly console interface

## Technologies Used

- Java
- Scanner Class
- Arrays

## How to Run

### Compile

```bash
javac Tic_Tac_Toe.java
```

### Run

```bash
java Tic_Tac_Toe
```
